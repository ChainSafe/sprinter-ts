## Supported Bridges

Sprinter currently supports the following bridging protocols:
- [LI.FI](https://li.fi/)
- [Across](https://across.to/)
- [Sygma](https://buildwithsygma.com)

## Supported Networks

Sprinter currently supports the following blockchain networks:
- Ethereum Mainnet
- Base
- Arbitrum
- Polygon
- Gnosis
- Cronos
